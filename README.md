## Abstract ##

nah karena kami diberikan 5 pilihan untuk mebuat sebuah web sederhana, dari pilihan yg telah diberikan.

Your webpage can be about any one topic below:

- student profile page
- favorite hobby page
- mini portfolio page
- course information page
- simple product or book showcase

karena saya membuat portfolio sederhana, saya memilih mini portfolio page. 
temanya sendiri itu mengenai Photography portfolio,

## Fitur ##

| Fitur | Deskripsi |
|-------|-----------|
|  **Profile Card** | Menampilkan nama, spesialisasi, pengalaman, dan foto profil |
|  **Quote Toggle** | Tombol "Show/Hide Quote" - menampilkan kutipan fotografer |
|  **Camera Gear** | Tombol "Show/Hide Camera Gear" - menampilkan atau menyembunyikan daftar alat kamera |
|  **Recent Projects** | Tabel 3 project terbaru dengan lokasi dan tahun |
|  **CSS Styling** | Desain modern dengan background gradient dan efek glassmorphism |
|  **Responsive** | Tampilan responsif untuk desktop dan mobile |
